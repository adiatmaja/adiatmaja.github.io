ASAH - Waste Classification App

Team Project

Key Highlights

  • Boosted Classification Accuracy to 94% (a 6.1% improvement) by replacing the VGG16 model with the more efficient MobileNetV2 architecture.
  • Enabled Real-Time Mobile Inference by optimizing the model for low-latency performance, leading to a successful integration in the production Android app.

Tech Stack

Python

Contributions

1. Model Architecture Optimization

The Challenge

The initial VGG16 model was too resource-intensive and not accurate enough for a responsive mobile waste classification application.

My Solution

I researched and implemented MobileNetV2, a lightweight architecture designed for mobile devices. This strategic change not only improved classification accuracy by 6.1% to a final 94%, but also significantly reduced the model's size and computational cost.

Model Performance: VGG16 vs. MobileNetV2

A comparison of accuracy and inference speed between the two models.

2. Mobile Deployment and Integration

The Challenge

The final model needed to be seamlessly integrated into a production Android application and perform classification in real-time on a mobile device's camera feed.

My Solution

By leveraging the efficient convolutions of MobileNetV2 and applying post-training quantization, I created a highly optimized model file. This ensured smooth, real-time inference on the device, allowing users to get instant feedback on waste classification and contributing to a successful app launch.
