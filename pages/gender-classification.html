Gender Classification

Fingerprint Analysis Thesis

This research project focused on exploring the effectiveness of Gabor filters in extracting features from fingerprint images for gender classification. The study aimed to determine if enhancing fingerprint patterns could improve the accuracy of machine learning models in identifying gender.

Research Methodology

The project involved a comparative analysis between raw fingerprint images and those processed with Gabor filters. We experimented with various Gabor filter parameters (frequency, orientation) to find the optimal configuration for feature enhancement.

Results

  • Performance Boost: Achieved a 7% improvement in F1-score when using Gabor-filtered images compared to the baseline.
  • Feature Extraction: Demonstrated that Gabor filters effectively highlight ridge structures that are discriminative for gender classification.
  • Model: Utilized Convolutional Neural Networks (CNN) for the classification task, implemented in Python using TensorFlow/Keras.

Impact

This thesis contributes to the field of biometrics by providing empirical evidence on the utility of texture analysis techniques like Gabor filtering for soft biometric traits classification.
