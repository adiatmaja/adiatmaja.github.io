Data Scientist

Sigma Solusi Indonesia

At Sigma Solusi Indonesia, I serve as a Data Scientist focusing on supply chain optimization and intelligent system development. My primary responsibility is to leverage data to improve operational efficiency and decision-making processes.

Key Responsibilities & Achievements

  • Demand Forecasting: Designed and implemented end-to-end forecasting pipelines using PyTorch and SQL. The models predict demand and sales trends, enabling better inventory planning and reducing stockouts.
  • Stock Balancing: Developed a stock balancing system that utilizes sales forecasts and warehouse simulations. This system optimizes the distribution of goods across the warehouse network, minimizing holding costs and transfer times.
  • RAG System Development: Built a Retrieval-Augmented Generation (RAG) system using ChromaDB. I implemented knowledge distillation techniques to improve retrieval accuracy and automated the entire workflow using n8n, making organizational knowledge more accessible.

Technologies Used

I rely heavily on Python and SQL for data manipulation and modeling. For deep learning tasks, I utilize PyTorch. The RAG system involves vector databases (ChromaDB) and workflow automation tools (n8n).
