Gender Classification from Fingerprints

Personal Project

Key Highlights

  • Improved F1-Score by 7% over baseline models by implementing Gabor filters for critical fingerprint feature enhancement.
  • Deployed as an Interactive Web App using Streamlit, providing an accessible interface for real-time model validation and demonstration.

Tech Stack

TensorFlow

Python

Contributions

1. Fingerprint Feature Enhancement

The Challenge

Raw fingerprint images lacked the distinct features needed for the deep learning model to accurately classify gender.

My Solution

I implemented Gabor filters as a crucial preprocessing step. This technique enhanced the textural features of the fingerprints, such as ridge frequency and orientation, providing the model with richer information and boosting the F1-score by 7%.

Before & After Gabor Filter

Comparison of a fingerprint image before and after applying the Gabor filter.

2. Interactive Streamlit Web App

The Challenge

The finished model needed an accessible and user-friendly platform for demonstration and real-time validation by non-technical users.

My Solution

I developed a web application using Streamlit that provides a simple interface to upload a fingerprint image. The app processes the image, runs the classification model, and displays the predicted gender instantly, making the project's outcome tangible and easy to share.
